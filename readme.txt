1.はじめに
	ゲームボーイアドバンス上において、
	お手軽に演奏を行うためのプログラムです。
	ゲーム機の音を自在に操る楽しさを味わってください。


2.操作方法
	- 十字キー押し続け	+ AorB		... 音を鳴らす
	- 十字キー押し続け     	+ START		... キー変更
	- START押し続け		+ 十字↑↓	... オクターブ変更
	- START押し続け		+ 十字←→	... 音変更(Duty比)

	※ キー変更について
	- キー:00 ... ←↖ 押し続け + START
	- キー:01 ... ←  押し続け + START
	- キー:02 ... ←↙ 押し続け + START

	- キー:03 ... ↓↙ 押し続け + START
	- キー:04 ... ↓  押し続け + START
	- キー:05 ... ↓ ↘押し続け + START

	- キー:06 ... → ↘押し続け + START
	- キー:07 ... →  押し続け + START
	- キー:08 ... → ↗押し続け + START

	- キー:09 ... ↑ ↗押し続け + START
	- キー:10 ... ↑  押し続け + START
	- キー:11 ... ↑↖ 押し続け + START


3.操作例
	- 「ド」を弾きたい場合。
	... 「 ↘押し続けB」と入力。

	- 方向と数字の対応
	-- ミ   ... ↖ B
	-- ファ ... ↖ A
	-- ソ   ... ←A
	-- ラ   ... ↙ A
	-- シ   ... ↓A
	-- ド   ...  ↘B
	-- レ   ... →B
	-- ミ   ...  ↗B
	-- ファ ...  ↗A
	-- ソ   ... ↑A
	-- ラ   ... ↖ A

	※ 一番低い・高い音で使用する"↖ " は、 
	   "←↖ " と押すか "↑↖ " と押すかで使い分けます。
	※すべて"キー : 4" の場合
